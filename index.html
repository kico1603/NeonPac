CONFIG FIREBASE

Para que el juego conecte por datos móviles, WiFi y cualquier red sin TURN flaky, usamos un proyecto Firebase gratis como relay.

  1. Entra a console.firebase.google.com (necesitas cuenta Google)
  2. Crea un proyecto nuevo. Cualquier nombre. Salta Google Analytics
  3. Menú izquierdo: Build → Realtime Database → Create Database
  4. Región: europe-west1 (Europa) o la más cercana
  5. Reglas: "Start in test mode"
  6. Vuelve al panel: ⚙ Project settings → "Your apps" → icono web </>
  7. Registra una app (cualquier nick), copia el bloque firebaseConfig = { ... } entero y pégalo aquí abajo
Importante: tu amigo también necesita pegar la misma config en su navegador. Compártesela por WhatsApp (es pública, no es secreto).
